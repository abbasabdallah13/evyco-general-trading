# Evyco

## Overview
This project is a company portfolio that showcases an overview about a company and its services. It is built using next.js.

## Features
- Landing page
- Products page
- Contact section
- Responsive design

## Demo
You can view a live demo of this project [here](https://evyco.net).

## Technologies Used
- ReactJS
- NextJS
